Privacy Policy

Last updated: July 2026

Overview

KI Manager Content Bot is a personal application run locally on the user's device. It is not a public web service and does not collect data from other users.

Data we process

  • TikTok OAuth tokens stored locally in a .env file on the user's machine
  • Discord messages sent to the user's own bot instance
  • Generated video files stored locally in a content/ folder

Third-party services

  • TikTok API — video publishing after user approval
  • Pixabay API — royalty-free stock footage
  • OpenAI / Tavily — content research (user's own API keys)

Data sharing

We do not sell, rent, or share personal data with third parties for marketing purposes.

User control

The user can revoke TikTok access at any time in TikTok account settings and delete local files and API keys at any time.
